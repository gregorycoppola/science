# Kant's Synthetic A Priori Is Lakatos's Hard Core

## The Two Ideas

**Kant's synthetic a priori**: certain propositions are both informative (not true by definition) and necessarily true (known prior to experience, holding for any possible experience). Kant's examples: "Every event has a cause." "Space is Euclidean." "Time is linear and absolute." These are not learned from experience — they are preconditions of experience, the framework the mind imposes on raw sensory input in order to have experience at all. They cannot be revised, because revising them would make experience impossible.

**Lakatos's hard core**: every research programme has a set of fundamental assumptions that are treated as irrefutable by methodological convention. The hard core is protected from falsification: when experiments produce anomalies, scientists modify the protective belt of auxiliary hypotheses, not the hard core. The hard core is not immune from revision in principle — a programme can be abandoned entirely — but within an active programme, it functions as fixed.

These are the same idea. The difference is one of scope and revisability.

## The Identification

Kant's synthetic a priori propositions play exactly the role of Lakatos's hard core within the research programme of Newtonian science:

- "Every event has a cause" — hard core commitment of classical mechanics.
- "Space is Euclidean" — hard core of Newtonian physics and classical geometry.
- "Time is absolute and universal" — hard core of Newtonian mechanics.

Scientists working within Newtonian mechanics did not question these. They were the framework within which problems were posed and solved. Anomalies were explained by modifying auxiliary hypotheses — not by questioning absolute time or Euclidean space. These commitments functioned as synthetic a priori: informative (they rule out non-Euclidean spaces, relative simultaneity), and treated as unrevisable within the programme.

## The Critical Difference

Kant thought synthetic a priori propositions were necessarily and permanently true — preconditions of any possible experience whatsoever. Lakatos recognized that hard core commitments are revisable in principle: a research programme can be abandoned, and with it the hard core.

Non-Euclidean geometry (Riemann, 1854) and special relativity (Einstein, 1905) falsified Kant's specific candidates for synthetic a priori status. Space is not necessarily Euclidean — Riemannian geometry is consistent and physically applicable. Time is not necessarily absolute — relativistic simultaneity is observer-dependent.

Lakatos provides the correct update on Kant: hard core commitments function as synthetic a priori within a programme, but they are abandoned when the programme is abandoned. They are not preconditions of all possible experience — they are preconditions of experience as structured by a particular research programme. When a better programme comes along, the old preconditions are revised.

## Lakatos Is Kant, Empiricized

The Kantian insight — that some commitments function as frameworks rather than hypotheses, structuring inquiry rather than being tested by it — is correct. The error was making these commitments permanent.

Lakatos empiricizes Kant: the hard core plays the Kantian role (it is not directly tested, it structures what counts as a problem and a solution) but it is contingent (it can be abandoned when a better programme offers a better hard core).

This is a precise philosophical improvement on Kant, achieved not by abstract argument but by attending to the actual structure of scientific history. The move from Kant to Lakatos is the same move as from the synthetic a priori to the methodologically protected hard core — from "cannot be revised" to "is not revised within this programme, but can be in a revolution."

## The MDL Gloss

In MDL terms: the hard core is the part of L(H) that is fixed across all theories within a programme. It is the common prefix of all theories in the hypothesis class. Modifying the hard core means switching hypothesis classes — a paradigm shift, which is a large discrete jump in the hypothesis space.

The synthetic a priori is the illusion that this common prefix is permanently fixed. Lakatos correctly identifies it as a programme-relative constant, not a universal one. The Bayesian/Kolmogorov framework makes this precise: the hard core is part of L(H), and if a competing programme has a different (shorter, or more compressive) hard core, the rational move is to switch.
