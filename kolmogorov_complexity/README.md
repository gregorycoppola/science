# Kolmogorov Complexity

This directory is a self-contained treatment of Kolmogorov complexity for a scientific audience. The goal is not a textbook — it is to convey the *weight* of the concept: why it matters, what it actually says, and how it connects to the question of what science is doing.

## Contents

- `INTUITION.md` — what Kolmogorov complexity is, built up from examples before any formalism
- `FORMAL.md` — the precise definition, key theorems, and what they imply
- `INVARIANCE.md` — the invariance theorem: why the choice of machine doesn't matter (up to a constant)
- `UNCOMPUTABILITY.md` — why K is uncomputable, and why that is a feature not a bug
- `MDL.md` — Minimum Description Length: the computable approximation you can actually use
- `SOLOMONOFF.md` — how Kolmogorov complexity generates the optimal prior for science
- `CONNECTIONS.md` — connections to entropy, compression, randomness, and learning theory
