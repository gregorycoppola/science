# Talks

Transcripts and analyses of talks that bear directly on the thesis of this repo —
that science has an objective function, and that function is `L(H) + L(D|H)`.

Each talk gets its own directory containing the transcript, a close reading, key
passages organized by theme, and explicit connections to the rest of the repo.

## Contents

| directory | talk | why it matters |
|-----------|------|----------------|
| `ilya_generalization/` | Ilya Sutskever, "An Observation on Generalization" (Simons Institute, Aug 2023) | The co-inventor of the transformer arguing that compression is the foundation of unsupervised learning — the same thesis as this repo, from the ML side |

## The Pattern

Every talk in this directory is making the same argument from a different angle:
compression, prediction, and generalization are the same thing. Kolmogorov
complexity is the unifying framework. The history of ML is the history of
approaching the Kolmogorov compressor from below.

The `science` repo makes this argument from philosophy and information theory.
These talks make it from empirical ML. The convergence is the point.
